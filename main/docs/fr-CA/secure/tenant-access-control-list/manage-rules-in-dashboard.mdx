Vous pouvez utiliser Auth0 Dashboard pour afficher la liste des règles ACL de votre tenant et les activer ou les désactiver individuellement.

<div id="view-list-of-rules">
  ## Afficher la liste des règles
</div>

Accédez à [Dashboard &gt; Security &gt; Access Control](https://manage.auth0.com/#/security/access-control) pour afficher la liste de vos règles existantes.

| Nom de la colonne | Description | Champ de réponse d’API correspondant |
| --- | --- | --- |
| Priority | Numéro ordinal indiquant à quel moment la règle est évaluée | `priority` |
| Name | Description de l’objectif ou de la fonctionnalité de la règle | `description` |
| Action | L’[action](/docs/fr-CA/secure/tenant-access-control-list/reference#actions) que la règle exécute. | `action` |
| Status | Valeur indiquant si la règle est active ou désactivée. | `active` |
| Last Updated | Horodatage de la dernière mise à jour de la règle. | `updated_at` |

![Afficher la liste des règles ACL de tenant dans l’Auth0 Dashboard](/docs/images/cdy7uua7fh8z/4093f9a39aba48bdf8d579c134dc4da5/Screenshot_2025-09-03_at_12.33.47.png)

<div id="enable-or-disable-a-rule">
  ## Activer ou désactiver une règle
</div>

Vous pouvez activer ou désactiver une règle spécifique en basculant le bouton situé à côté de celle-ci.

Le champ **Status** indique si la règle est `Active` ou `Disabled`.
