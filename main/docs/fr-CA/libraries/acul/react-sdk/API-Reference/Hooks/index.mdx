<div id="methods">
  ## Méthodes
</div>

* [useAuth0Themes](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes): Hook React pour obtenir les options de thème actuelles avec une configuration aplatie provenant du contexte d’image de marque.
* [useCurrentScreen](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen): Hook React qui obtient le contexte et l’état de l’écran actuel.
* [useErrors](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useErrors): Hook React pour lire et gérer les erreurs dans ACUL (Advanced Customization of Universal Login). Inclut toutes les erreurs de validation et côté serveur. Il regroupe les erreurs en trois catégories : - `server` — erreurs renvoyées par Auth0 ou par votre propre backend. - `client` — erreurs issues de la validation côté client (p. ex., saisie de formulaire non valide). - `developer` — erreurs causées par une intégration incorrecte ou une mauvaise utilisation du SDK.
* [useLoginIdentifiers](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useLoginIdentifiers): Retourne une liste de types d’identifiants actifs (comme `'email'`, `'phone'` ou `'username'`) actuellement utilisés dans le flux d’authentification ou le processus de connexion.
* [useMfaPolling](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling): Hook React pour gérer le polling des notifications push AMF (MFA) (p. ex., en attente de l’approbation d’une notification push) sur un écran Auth0 Advanced Customization of Universal Login (ACUL).
* [useResend](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useResend): Ce hook React gère les actions de renvoi (« resend ») (p. ex., renvoyer un code de vérification) sur les écrans ACUL.
* [useSignupIdentifiers](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useSignupIdentifiers): Retourne une liste d’identifiants activés (courriel, téléphone ou nom d’utilisateur), chacun avec son statut `required`, en fonction des identifiants d’inscription de l’écran actuel.
* [usePasswordValidation](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation): Ce hook React valide un mot de passe par rapport à la politique de mot de passe Auth0 actuelle et retourne un résultat structuré décrivant si le mot de passe satisfait à chaque règle.
* [useUsernameValidation](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useUsernameValidation): Hook React pour valider un nom d’utilisateur par rapport à la politique de nom d’utilisateur Auth0 actuelle.
