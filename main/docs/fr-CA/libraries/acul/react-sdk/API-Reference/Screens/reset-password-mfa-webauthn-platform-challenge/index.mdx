<ParamField body='rememberDevice?' type='boolean'>

Optionnel. Si `true` et que `screen.showRememberDevice` est aussi `true`,
                                       cela indique que l'utilisateur a choisi d’enregistrer le navigateur actuel,
                                       ce qui peut permettre de contourner l’AMF pendant une certaine période. Correspond au champ de formulaire `rememberBrowser`.
