---
title: "EmailOTPChallengeMembers"
---

Représente l’interface de l’écran de vérification OTP (mot de passe à usage unique, One-Time Password) par courriel.

```ts Example
export interface EmailOTPChallengeMembers extends BaseMembers {
  screen: ScreenMembersOnEmailOTPChallenge;
  /**
   * Soumet le code OTP saisi par l'utilisateur.
   * @param options Paramètres facultatifs à inclure dans la soumission.
   */
  submitCode(options: OtpCodeOptions): Promise<void>;
  /**
   * Demande l'envoi d'un nouveau code OTP au courriel de l'utilisateur.
   * @param options Paramètres facultatifs à inclure dans la demande de renvoi.
   */
  resendCode(options?: CustomOptions): Promise<void>;
  /**
   * Obtient la fonctionnalité de renvoi avec gestion du délai d'expiration pour cet écran
   * @param options Options de configuration pour la fonctionnalité de renvoi
   */
  resendManager(options?: StartResendOptions): ResendControl;
}
```
