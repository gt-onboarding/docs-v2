---
title: "ContextHooks<T>"
---

Classe de fabrique permettant de créer des hooks de contexte pour accéder aux données d&#39;Universal Login d&#39;Auth0.
Ces hooks permettent d&#39;accéder à différents aspects du flux d&#39;authentification et de la configuration du tenant.

<div id="methods">
  ## Méthodes
</div>

<ParamField body="useUser()" type="T[&#x22;user&#x22;]">
  Hook pour accéder aux informations sur l’utilisateur et aux données de profil.

  Objet utilisateur contenant les informations de profil, les attributs et les données spécifiques à l’utilisateur

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/login-id';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useTenant()" type="T[&#x22;tenant&#x22;]">
  Hook pour accéder à la configuration et aux paramètres du tenant.

  Objet tenant contenant le domaine, la région et la configuration propre au tenant

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/login-id';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useBranding()" type="T[&#x22;branding&#x22;]">
  Hook pour accéder à la configuration de l’image de marque et du thème.

  Objet de branding contenant les couleurs, les logos, les polices et les paramètres de personnalisation visuelle

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/login-id';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient()" type="T[&#x22;client&#x22;]">
  Hook pour accéder à la configuration de l’Application Auth0 (client).

  Objet client contenant les paramètres de l’Application, les callbacks et les données propres au client

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/login-id';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization()" type="T[&#x22;organization&#x22;]">
  Hook pour accéder au contexte et aux paramètres d’organisation.

  Objet organisation contenant des données propres à l’organisation, des métadonnées et la configuration

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/login-id';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>Aucun contexte d’organisation</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt()" type="T[&#x22;prompt&#x22;]">
  Hook pour accéder à la configuration du prompt et aux paramètres du flux.

  Objet prompt contenant la configuration du flux, les paramètres d’écran et les données propres au prompt

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/login-id';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData()" type="T[&#x22;untrustedData&#x22;]">
  Hook pour accéder aux données non fiables provenant des paramètres d’URL et des envois de formulaires.

  Objet contenant les entrées non fiables de l’utilisateur qui doivent être validées avant utilisation

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/login-id';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useScreen()" type="T[&#x22;screen&#x22;]">
  Hook pour accéder aux informations et aux métadonnées de l’écran actuel.

  Objet écran contenant le nom de l’écran actuel, la configuration et les données propres à l’écran

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/login-id';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTransaction()" type="T[&#x22;transaction&#x22;]">
  Hook pour accéder à l’état de la transaction et aux données du flux d’authentification.

  Objet transaction contenant l’état du flux, les données de session et les informations propres à la transaction

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/login-id';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>
