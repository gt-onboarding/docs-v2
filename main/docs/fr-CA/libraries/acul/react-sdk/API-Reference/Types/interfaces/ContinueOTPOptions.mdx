---
title: "Options pour poursuivre le défi OTP"
---

Options permettant de poursuivre le défi One-Time Password (OTP)

```ts Example
export interface ContinueOptions {
  /** Le code entré par l'utilisateur */
  code: string;
  /** Indique s'il faut se souvenir du navigateur */
  rememberDevice?: boolean;
  /** Toutes options personnalisées supplémentaires */
  [key: string]: string | number | boolean | undefined;
}
```


<div id="indexable">
  ## Indexable
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`

Toute option personnalisée supplémentaire
