title: "OrganizationSelectionMembers"

Interface qui définit les méthodes et propriétés disponibles pour l&#39;écran de sélection d&#39;une organisation.

```ts Example
export interface OrganizationSelectionMembers extends BaseMembers {
  /**
   * Les propriétés de l'écran.
   */
  screen: ScreenMembersOnOrganizationSelection;
  /**
   * Poursuit avec le nom de l'organisation sélectionnée.
   * @param payload Les options contenant le nom de l'organisation.
   */
  continueWithOrganizationName(payload: ContinueWithOrganizationNameOptions): Promise<void>;
}
```
