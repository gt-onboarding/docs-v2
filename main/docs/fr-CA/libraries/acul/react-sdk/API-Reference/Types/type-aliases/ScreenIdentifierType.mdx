<ParamField body="ScreenIdentifierType" type='"email" | "phone" | "username"'>
  

Représente les types d'identifiants d'utilisateur qui peuvent être utilisés lors de l'inscription.

- `'email`' : L'utilisateur fournira une adresse de courriel.
- `'phone'` : L'utilisateur fournira un numéro de téléphone.
- `'username'` : L'utilisateur choisira un nom d'utilisateur.

Ce type est généralement utilisé dans les flux de sélection d'identité ou d'inscription
afin de définir quels identifiants sont activés et comment ils se comportent.

```ts Exemple
const type: IdentifierType = 'email';
```

</ParamField>
