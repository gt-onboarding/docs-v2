ContinueOptions
Définit les options nécessaires pour poursuivre le processus de défi téléphonique AMF (MFA).
Cela implique généralement de sélectionner la méthode (SMS ou appel vocal) pour recevoir le code de vérification.

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * Le type de défi à effectuer.
   * 'sms' : Envoie le code par message texte.
   * 'voice' : Envoie le code par appel vocal.
   * @default 'sms'
   */
  type: 'sms' | 'voice';
}
```

Le type de challenge à effectuer.
- `sms` : Envoyer le code par SMS.
- `voice` : Envoyer le code par appel vocal.

  #### Valeur par défaut
