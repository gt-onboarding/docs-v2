Paramètres facultatifs pour [useResend](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useResend).

<div id="properties">
  ## Propriétés
</div>

<ParamField body="timeoutSeconds?" type="number" default="10">
  Durée du compte à rebours (en secondes) avant qu’un autre renvoi soit autorisé.
  La valeur par défaut est `10`.
</ParamField>

<ParamField body="onTimeout()?" type="void">
  Fonction de rappel appelée lorsque le compte à rebours se termine et que l’action de renvoi devient de nouveau disponible.
</ParamField>
