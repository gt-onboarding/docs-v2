Objet de résultat renvoyé par [useMfaPolling](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling).

<div id="properties">
  ## Propriétés
</div>

<ParamField body="isRunning" type="boolean">
  Indique si le processus de sondage de l’AMF (MFA) push est actuellement actif.

  * `true` — Le sondage est en cours et en attente de finalisation.
  * `false` — Le sondage s’est arrêté, soit en raison de la finalisation,
    de l’annulation manuelle ou du démontage du composant.
</ParamField>

<ParamField body="startPolling()" type="void">
  Démarre ou reprend le processus de sondage.

  * Si le sondage est déjà actif, cet appel n’a aucun effet.
  * S’il a déjà été arrêté, appeler cette fonction redémarre la boucle de sondage.
</ParamField>

<ParamField body="stopPolling()" type="void">
  Arrête immédiatement le processus de sondage.

  * Annule tout minuteur programmé ou toute requête en cours.
  * Peut être appelée plusieurs fois sans danger; les appels suivants n’ont aucun effet.
</ParamField>
