TryAnotherMethodOptions

```ts Example
export interface TryAnotherMethodOptions extends CustomOptions {
  /**
   * La méthode de livraison qui a été initialement présentée ou sélectionnée sur l'écran actuel.
   * Ceci est requis par le point de terminaison de l'API pour traiter correctement l'action « pick-authenticator ».
   * - `sms` : Indique que l'écran actuel était destiné au défi SMS.
   * - `voice` : Indique que l'écran actuel était destiné au défi vocal.
   * @type {'sms' | 'voice'}
   */
  type: 'sms' | 'voice';
}
```
