Détails d&#39;une erreur du navigateur liée à WebAuthn.

```ts Example
export interface WebAuthnErrorDetails {
  /**
   * Le nom de l'erreur (p. ex. « NotAllowedError »).
   */
  name: string;
  /**
   * Le message d'erreur.
   */
  message: string;
  /**
   * Toutes propriétés d'erreur supplémentaires.
   */
  [key: string]: string | number | boolean | undefined;
}
```
