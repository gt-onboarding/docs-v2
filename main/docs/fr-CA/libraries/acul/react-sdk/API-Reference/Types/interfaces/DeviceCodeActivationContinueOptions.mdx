title: "DeviceCodeActivationContinueOptions"
---

Charge utile pour poursuivre l’activation du code de l’appareil.

Cette interface étend `CustomOptions`, ce qui signifie que toutes les propriétés
d’options personnalisées sont fusionnées au niveau supérieur de l’objet. Cela évite
l’imbrication et offre une structure plus simple pour les utilisateurs de cette interface.

```ts Example
export interface ContinueOptions extends CustomOptions {
  code: string;
}
```
