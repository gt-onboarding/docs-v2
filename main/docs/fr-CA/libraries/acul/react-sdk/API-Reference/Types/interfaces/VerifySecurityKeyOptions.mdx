<ParamField body='rememberDevice?' type='boolean'>
Facultatif. S’il est défini sur true, tente d’enregistrer le navigateur pour de futures demandes d’authentification multifacteur (AMF, MFA).
Correspond au champ de formulaire `rememberBrowser`.
