---
title: "MfaOtpContinueOptions"
---

Options permettant de poursuivre avec le code OTP d’inscription à l’AMF (MFA).

```ts Example
export interface ContinueOptions {
  /**
   * Le code OTP entré par l'utilisateur.
   */
  code: string;

  /**
   * Toutes options personnalisées supplémentaires.
   */
  [key: string]: string | number | boolean | undefined;
}
```


<div id="indexable">
  ## Indexable
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`

Toute option personnalisée supplémentaire.

<div id="properties">
  ## Propriétés
</div>

<ParamField body='code' type='string'>
Le code OTP saisi par l’utilisateur.
</ParamField>
