VerifyPlatformAuthenticatorOptions

```ts Example
export interface VerifyPlatformAuthenticatorOptions extends CustomOptions {
  /**
   * Facultatif. Si la valeur est true et que `screen.showRememberDevice` est true, cela indique que l'utilisateur
   * a choisi de mémoriser cet appareil, généralement pour 30 jours, afin de contourner l'AMF (MFA) lors des connexions subséquentes.
   * @type {boolean}
   */
  rememberDevice?: boolean;
}
```

<ParamField body='rememberDevice?' type='boolean'>
Facultatif. Si la valeur est true, le navigateur sera mémorisé pour de futurs défis d’AMF (MFA).
Correspond au champ de formulaire `rememberBrowser`. Ceci ne s’applique que si `screen.showRememberDevice` est true.
</ParamField>
