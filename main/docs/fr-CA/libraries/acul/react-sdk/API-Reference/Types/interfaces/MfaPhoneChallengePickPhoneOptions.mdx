PickPhoneOptions
Définit les options pour l’action &#39;pick-phone&#39;.
Même si l’interface n’affiche qu’un seul numéro de téléphone, cette action peut être utilisée
dans des scénarios où le serveur doit rediriger vers un écran de sélection d’un téléphone.

```ts Example
export interface PickPhoneOptions extends CustomOptions {}
```
