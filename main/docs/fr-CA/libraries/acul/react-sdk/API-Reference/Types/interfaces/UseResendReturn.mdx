Type de retour du hook [useResend](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Hooks/useResend).

<div id="properties">
  ## Propriétés
</div>

<ParamField body="remaining" type="number">
  Nombre de secondes restantes avant que la prochaine tentative de renvoi soit autorisée.
</ParamField>

<ParamField body="disabled" type="boolean">
  Indique si l’action de renvoi est actuellement désactivée.
</ParamField>

<ParamField body="startResend()" type="void">
  Lance immédiatement une tentative de renvoi, si elle est autorisée.
</ParamField>
