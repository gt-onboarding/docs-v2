title: "MfaPushListMembers"
---

Interface définissant les méthodes et propriétés disponibles pour l&#39;écran `mfa-push-list`

```ts Example
export interface MfaPushListMembers extends BaseMembers {
  /**
   * Sélectionne un appareil enregistré dans la liste pour lancer une notification push AMF (MFA).
   * @param payload Les options contenant l'index de l'appareil.
   */
  selectMfaPushDevice(payload: SelectMfaPushDeviceOptions): Promise<void>;
  /**
   * Navigates back to the previous screen.
   * @param payload Optional custom options to include with the request.
   */
  goBack(payload?: CustomOptions): Promise<void>;
}
```
