ContinueWithCodeOptions
description Options pour la méthode `continueWithCode` de l&#39;écran de vérification du courriel de connexion.
Cette opération est utilisée lorsque l&#39;utilisateur saisit le code de vérification qu&#39;il a reçu par courriel
pour poursuivre le processus d&#39;authentification.

```ts Example
export interface ContinueWithCodeOptions extends CustomOptions {
  /**
   * Le code de vérification envoyé au courriel de l'utilisateur. Il s'agit généralement d'une courte chaîne numérique
   * ou alphanumérique que l'utilisateur doit entrer dans le formulaire.
   * @type {string}
   * @example "123456"
   * @example "ABCXYZ"
   */
  code: string;
}
```
