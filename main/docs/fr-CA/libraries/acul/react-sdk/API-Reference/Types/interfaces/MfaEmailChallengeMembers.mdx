Interface qui définit les méthodes et les propriétés disponibles pour l&#39;écran mfa-email-challenge

```ts Example
export interface MfaEmailChallengeMembers extends BaseMembers {
  screen: ScreenMembersOnMfaEmailChallenge;
  untrustedData: UntrustedDataMembersOnMfaEmailChallenge;

  /**
   * Poursuit le défi par courriel en utilisant le code fourni
   * @param payload Les options contenant le code et l'indicateur rememberDevice
   */
  continue(payload: ContinueOptions): Promise<void>;

  /**
   * Renvoie le code par courriel
   * @param payload Options personnalisées facultatives à inclure dans la requête
   */
  resendCode(payload?: ResendCodeOptions): Promise<void>;

  /**
   * Permet à l'utilisateur d'essayer une autre méthode AMF (MFA)
   * @param payload Options personnalisées facultatives à inclure dans la requête
   */
  tryAnotherMethod(payload?: TryAnotherMethodOptions): Promise<void>;

  /**
   * Soumet l'action pour sélectionner une configuration de courriel différente, si disponible
   * @param payload Options personnalisées facultatives à inclure dans la requête
   */
  pickEmail(payload?: CustomOptions): Promise<void>;

  /**
   * Obtient la fonctionnalité de renvoi avec gestion du délai d'expiration pour cet écran
   * @param options Options de configuration pour la fonctionnalité de renvoi
   */
  resendManager(options?: StartResendOptions): ResendControl;
}
```
