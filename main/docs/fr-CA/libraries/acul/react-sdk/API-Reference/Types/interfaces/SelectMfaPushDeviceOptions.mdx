---
title: "SelectMfaPushDeviceOptions"
---

Options permettant de sélectionner un appareil AMF (MFA) pour les notifications push.

```ts Example
export interface SelectMfaPushDeviceOptions {
  /**
   * L'index de l'appareil à sélectionner dans la liste des appareils inscrits.
   * L'index est basé sur 0.
   */
  deviceIndex: number;
  [key: string]: string | number | boolean | undefined;
}
```


<div id="indexable">
  ## Indexable
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`

<div id="properties">
  ## Propriétés
</div>

<ParamField body='deviceIndex' type='number'>
L’index de l’appareil à sélectionner dans la liste des appareils inscrits.
L’index commence à 0.
</ParamField>
