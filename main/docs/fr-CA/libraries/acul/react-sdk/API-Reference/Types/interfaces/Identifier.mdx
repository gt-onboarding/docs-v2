---
title: "Identifiant"
---

Décrit un champ d’identifiant configuré (par exemple courriel, téléphone ou nom d&#39;utilisateur)
pour un formulaire d’inscription ou un flux d’authentification.

```ts Example
export interface Identifier {
  /**
   * Le type d'identifiant, tel que `'email'`, `'phone'` ou `'username'`.
   *
   * @see IdentifierType
   */
  type: IdentifierType;

  /**
   * Whether this identifier is required for signup.
   */
  required: boolean;
}
```


<div id="example">
  ## Exemple
</div>

```ts
const emailIdentifier: Identifier = {
  type: 'email',
  required: true
};
```
