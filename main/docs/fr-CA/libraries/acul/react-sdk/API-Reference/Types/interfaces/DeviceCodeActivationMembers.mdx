---
title: "DeviceCodeActivationMembers"
---

Interface qui décrit les membres de l’écran d’activation du code d’appareil.

```ts Example
export interface DeviceCodeActivationMembers extends BaseMembers {
  /**
   * Soumet le code d'appareil saisi par l'utilisateur.
   * Cette action est déclenchée lorsque l'utilisateur saisit le code affiché sur son appareil et soumet le formulaire.
   *
   * @param {object} payload - Un objet contenant le code saisi par l'utilisateur et toute charge utile personnalisée.
   * @param {string} payload.code - Le code d'appareil saisi par l'utilisateur.
   *
   * @returns {Promise<void>} Une promesse qui se résout lorsque le code est soumis avec succès.
   * Est rejetée avec une erreur si la soumission échoue.
   */
  continue(payload: ContinueOptions): Promise<void>;
}
```
