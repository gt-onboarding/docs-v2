TryAnotherMethodOptions

```ts Example
export interface TryAnotherMethodOptions extends CustomOptions {
  /**
   * Facultatif. Si `true`, et si le contexte de l'écran (`screen.showRememberDevice`)
   * indique que la mémorisation de l'appareil est une option, cela signale l'intention
   * de l'utilisateur de se souvenir de ce navigateur/appareil.
   * @type {boolean | undefined}
   */
  rememberDevice?: boolean;
}
```
