---
title: "UsernameValidationError"
---

Représente une erreur individuelle de validation du nom d&#39;utilisateur retournée lors de la validation.

Chaque erreur comporte un `code` lisible par machine et un `message` lisible par un humain.

```ts Example
export interface UsernameValidationError {
  /**
   * Identifiant unique de la règle de validation qui a échoué.
   * 
   * Exemple : `"username-too-short"`, `"username-invalid-characters"`
   */
  code: string;

  /**
   * A human-readable description of the error.
   * 
   * Example: `"Username must be at least 3 characters long."`
   */
  message: string;

  /**
   * The field associated with the validation error, typically "username", "email", or "phone".
   */
  field: string;
}
```

  ## Propriétés
</div>

<ParamField body='code' type='string'>

Identifiant unique de la règle de validation ayant échoué.

</ParamField>

<ParamField body='field' type='string'>

Champ associé à l’erreur de validation, généralement « username », « email » ou « phone ».

</ParamField>

<ParamField body='message' type='string'>

Description de l’erreur lisible par un humain.
