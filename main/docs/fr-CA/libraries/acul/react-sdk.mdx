La trousse de développement logiciel (SDK) React ACUL d&#39;Auth0 vous permet de travailler avec la [personnalisation avancée d’Universal Login](/docs/fr-CA/customize/login-pages/advanced-customizations).

Elle simplifie l&#39;intégration des écrans d&#39;authentification (connexion, inscription, authentification sans mot de passe, inscription aux passkeys, etc.) dans vos applications web, en fournissant les outils nécessaires pour une mise en œuvre fluide.

Pour commencer, installez la trousse de développement logiciel (SDK) JS ACUL via npm :

```bash
npm install @auth0/auth0-acul-react
npm install react
```

Après avoir installé la trousse de développement logiciel (SDK), vous pouvez importer les classes d’écran pour les configurer.

Explorez les [écrans, types et interfaces du SDK React ACUL](/docs/fr-CA/libraries/acul/react-sdk/API-Reference/Screens/login).

Vous pouvez également commencer avec le [Démarrage rapide ACUL](/docs/fr-CA/customize/login-pages/advanced-customizations/quickstart).
