<div id="constructors">
  ## Constructeurs

<ParamField body="MfaWebAuthnError" type="Constructor">

Initialise une nouvelle instance de la classe `MfaWebAuthnError`.
Récupère le contexte nécessaire (écran, transaction, etc.) à partir du `universal_login_context` global.

<div id="throws">
  #### Exceptions
</div>

Si le contexte Universal Login n’est pas disponible ou si le nom de l’écran dans le contexte
ne correspond pas à `MfaWebAuthnError.screenIdentifier`.

</ParamField>

<ParamField body='noThanks' type='Promise<void>'>

Permet à l’utilisateur de refuser l’opération WebAuthn en cours (p. ex., refuser d’ajouter un appareil lors de l’inscription).
Cette action indique le choix de l’utilisateur de ne pas poursuivre l’étape WebAuthn pour le moment.

Une promesse résolue lorsque l’action « refuse-add-device » est envoyée avec succès.

#### Lève

Lève une erreur si l’envoi du formulaire échoue.

```typescript Example
// En supposant que 'sdk' est une instance de MfaWebAuthnError
await sdk.noThanks();
```

<ParamField body='tryAgain' type='Promise<void>'>

Permet à l’utilisateur de réessayer l’opération WebAuthn précédente.
Cette action redirige généralement l’utilisateur vers l’écran où le processus WebAuthn a été
initialement tenté (p. ex., retour à l’écran de défi d’authentification multifacteur (AMF, MFA) ou d’inscription pour WebAuthn).

Une promesse résolue lorsque l’action « tryagain » est envoyée avec succès.

#### Lève

Lève une erreur si l’envoi du formulaire échoue.

```typescript Example
// En supposant que 'sdk' est une instance de MfaWebAuthnError
await sdk.tryAgain();
```

<ParamField body='usePassword' type='Promise<void>'>

Permet à l’utilisateur de tenter l’authentification à l’aide de son mot de passe, si cette alternative est configurée
et appropriée pour le flux en cours (p. ex., si le mot de passe était une étape d’authentification précédente ou constitue une solution de contournement AMF valide).

Une promesse résolue lorsque l’action « use-password » est envoyée avec succès.

#### Lève

Lève une erreur si l’envoi du formulaire échoue.
