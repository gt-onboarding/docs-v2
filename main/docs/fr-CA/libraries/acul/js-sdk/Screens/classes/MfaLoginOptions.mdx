Classe implémentant la fonctionnalité de l&#39;écran mfa-login-options
Cet écran permet aux utilisateurs de sélectionner le facteur d’AMF (MFA) qu’ils souhaitent utiliser pour se connecter

```ts Example
const mfaLoginOptions = new MfaLoginOptions();
await mfaLoginOptions.enroll({
  action: 'push-notification'
});
```
