Classe qui implémente la fonctionnalité de l’écran d’activation du code de l’appareil.
Cet écran s’affiche lorsque l’utilisateur doit saisir le code sur son appareil pour l’activer.

```ts Example
import DeviceCodeActivation from '@auth0/auth0-acul-js/device-code-activation';
const deviceCodeActivationManager = new DeviceCodeActivation();
async function activateDeviceCode(code) {
  try {
   await deviceCodeActivationManager.continue({ code });
   console.log('Activation du code de l'appareil réussie.');
  } catch (error) {
   console.error('Erreur lors de l'activation du code de l'appareil :', error);
  }
}
```

  ## Méthodes
</div>

<ParamField body='continue' type='Promise<void>'>

Soumet le code d’appareil saisi par l’utilisateur.
Cette action est déclenchée lorsque l’utilisateur entre le code affiché sur son appareil et envoie le formulaire.

Une promesse résolue lorsque le code est soumis avec succès.

```typescript Example
import DeviceCodeActivation from '@auth0/auth0-acul-js/device-code-activation';
const deviceCodeActivationManager = new DeviceCodeActivation();
async function activateDeviceCode(code) {
  try {
   await deviceCodeActivationManager.continue({ code });
   console.log('Device code activation successful.');
  } catch (error) {
   console.error('Error during device code activation:', error);
  }
}
```
Rejetée avec une erreur si l’envoi échoue.
