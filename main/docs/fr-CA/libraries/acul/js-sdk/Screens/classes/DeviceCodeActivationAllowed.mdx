Classe qui implémente la fonctionnalité de l&#39;écran Device Code Activation Allowed.
Cet écran s&#39;affiche lorsque l&#39;activation du code de l&#39;appareil est autorisée.

```ts Example
import DeviceCodeActivationAllowed from "@auth0/auth0-acul-js/device-code-activation-allowed";
const deviceCodeActivationAllowedManager = new DeviceCodeActivationAllowed();
```
