---
title: "MfaCountryCodes"
---

Classe qui implémente la fonctionnalité de l&#39;écran mfa-country-codes
Cet écran permet aux utilisateurs de sélectionner un indicatif de pays pour la vérification du numéro de téléphone dans le cadre de l’AMF (MFA)

```ts Example
import MfaCountryCodes from '@auth0/auth0-acul-js/mfa-country-codes';

const mfaCountryCodes = new MfaCountryCodes();

// Obtenir les codes de pays et les préfixes de téléphone disponibles
const { screen } = mfaCountryCodes;
const { phone_prefixes } = screen.data
const {country_code, phone_prefix} = phone_prefixes[0]

await mfaCountryCodes.selectCountryCode({
  country_code: 'US',
  phone_prefix: '+1',
});
```
