Classe d’implémentation de l’écran d’options de début d’inscription à l’AMF (MFA)
Gère la sélection et l’inscription des facteurs d’AMF (MFA)

```ts Example
const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
await mfaBeginEnrollOptions.enroll({
  action: 'push-notification'
});
```
