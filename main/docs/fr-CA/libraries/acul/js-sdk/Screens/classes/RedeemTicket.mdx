  ## Méthodes

<ParamField body='continue' type='Promise<void>'>

Effectue l’action par défaut sur l’écran « redeem-ticket », généralement pour poursuivre le flux.

Une promesse qui se résout lorsque l’action est terminée.

<div id="throws">
  #### Lève
</div>

Si l’opération échoue.

```ts Example
import RedeemTicket from '@auth0/auth0-acul-js/redeem-ticket';

const redeemTicket = new RedeemTicket();
await redeemTicket.continue();
```
