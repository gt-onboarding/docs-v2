Représente l&#39;écran d&#39;inscription à l&#39;AMF (MFA) par SMS.

```ts Example
import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

const mfaSmsEnrollment = new MfaSmsEnrollment();
await mfaSmsEnrollment.pickCountryCode();
```

<ParamField body='continueEnrollment' type='Promise<void>'>

Poursuit le processus d’inscription par SMS avec le numéro de téléphone fourni.

Une promesse qui se résout lorsque le processus d’inscription est terminé.

<div id="throws">
  #### Lève
</div>

Si le numéro de téléphone est manquant.

```typescript Example
import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

const mfaSmsEnrollment = new MfaSmsEnrollment();
await mfaSmsEnrollment.continueEnrollment({ phone: '1234567890' });
```
