MfaEnrollResult

```typescript Example
import MfaEnrollResult from '@auth0/auth0-acul-js/mfa-enroll-result';

const mfaEnrollResultScreen = new MfaEnrollResult();

// Accéder aux données de l'écran
const enrollmentStatus = mfaEnrollResultScreen.screen.data?.status;
const pageTitle = mfaEnrollResultScreen.screen.texts?.title;

console.log(`Statut d'inscription AMF : ${enrollmentStatus}`);
console.log(`Titre de la page : ${pageTitle}`);
```


<div id="constructors">
  ## Constructeurs
</div>

<ParamField body="MfaEnrollResult" type="Constructor">

Crée une instance du gestionnaire d’écran `MfaEnrollResult`.
Ce constructeur initialise le `BaseContext` et configure la propriété `screen`
avec une instance de `ScreenOverride` adaptée à cet écran.

<div id="throws">
  #### Lève
</div>

Une exception si le contexte Universal Login n’est pas disponible ou si le
nom de l’écran actuel dans le contexte ne correspond pas à `MfaEnrollResult.screenIdentifier`.

</ParamField>
