Classe qui implémente la fonctionnalité de l&#39;écran de déconnexion.
Cet écran permet aux utilisateurs de confirmer ou de refuser la déconnexion.

```ts Example
import Logout from '@auth0/auth0-acul-js/logout';

const logout = new Logout();

await logout.confirmLogout({ action: 'accept' });
```

<ParamField body='confirmLogout' type='Promise<void>'>

Confirme l’action de déconnexion (accepter ou refuser).

```typescript Example
import Logout from '@auth0/auth0-acul-js/logout';

const logout = new Logout();

await logout.confirmLogout({ action: 'accept' });
```
