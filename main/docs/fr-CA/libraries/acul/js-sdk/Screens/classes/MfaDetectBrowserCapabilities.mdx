Classe qui implémente la fonctionnalité de l&#39;écran mfa-detect-browser-capabilities.
Cet écran détecte les capacités du navigateur pour les méthodes d&#39;authentification multifacteur (AMF, MFA).

```ts Example
const mfaDetectBrowserCapabilities = new MfaDetectBrowserCapabilities();
await mfaDetectBrowserCapabilities.detectCapabilities();
```
