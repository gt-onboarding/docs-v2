title: "MfaEmailList"
---

Classe qui implémente la fonctionnalité de l&#39;écran mfa-email-list
Cet écran permet aux utilisateurs de sélectionner une adresse de courriel inscrite pour l’AMF (MFA)

```ts Example
import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';

const mfaEmailList = new MfaEmailList();
await mfaEmailList.selectMfaEmail({
  index: 0 // à titre de démonstration, nous sélectionnons le premier index
});
```
