ContinueWithPasskeyOptions

```ts Example
export interface ContinueWithPasskeyOptions extends CustomOptions {
  /**
   * Facultatif. Si la valeur est true et si `screen.showRememberDevice` est également true,
   * cela indique au serveur de mémoriser ce navigateur pour l'AMF (MFA) pendant une période définie.
   * @type {boolean | undefined}
   */
  rememberDevice?: boolean;
}
```

<ParamField body='rememberDevice?' type='boolean'>
Facultatif. Si la valeur est `true` et que `screen.showRememberDevice` est également défini sur `true`,
cela indique que l'utilisateur a choisi de mémoriser le navigateur actuel,
ce qui peut permettre de contourner l’AMF (MFA) pendant une certaine période. Correspond au champ de formulaire `rememberBrowser`.
