---
title: "MfaOtpContinueOptions"
---

Options permettant de poursuivre l&#39;inscription à l&#39;AMF (MFA) avec le code OTP (One-Time Password).

```ts Example
export interface ContinueOptions {
  /**
   * Le code OTP entré par l'utilisateur.
   */
  code: string;

  /**
   * Toutes options personnalisées supplémentaires.
   */
  [key: string]: string | number | boolean | undefined;
}
```


<div id="indexable">
  ## Indexable
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`

Toute autre option personnalisée.

<div id="properties">
  ## Propriétés
</div>

<ParamField body='code' type='string'>

Le code One-Time Password (OTP) saisi par l'utilisateur.
</ParamField>
