---
title: "ContinuePayloadOptions"
---

Options permettant de poursuivre le défi par courriel

```ts Example
export interface ContinueOptions {
  /** Le code entré par l'utilisateur */
  code: string;
  /** Indique si l'appareil doit être mémorisé */
  rememberDevice?: boolean;
  /** Toutes options personnalisées supplémentaires */
  [key: string]: string | number | boolean | undefined;
}
```


<div id="indexable">
  ## Indexable
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`

Toute autre option personnalisée supplémentaire
