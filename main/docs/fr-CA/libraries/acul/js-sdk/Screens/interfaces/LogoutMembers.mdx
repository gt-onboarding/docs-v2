title: "LogoutMembers"

Interface qui définit les méthodes et propriétés disponibles pour l&#39;écran de déconnexion

```ts Example
export interface LogoutMembers extends BaseMembers {
  /**
   * Confirme l'action de déconnexion (accepter ou refuser)
   * @param payload Les options contenant l'action à exécuter
   */
  confirmLogout(payload: ConfirmLogoutOptions): Promise<void>;
}
```
