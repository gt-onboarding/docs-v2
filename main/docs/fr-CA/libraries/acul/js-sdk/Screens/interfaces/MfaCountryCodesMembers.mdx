title: "MfaCountryCodesMembers"
---

Interface qui définit les méthodes et les propriétés disponibles pour l&#39;écran `mfa-country-codes`

```ts Example
export interface MfaCountryCodesMembers extends BaseMembers {
  screen: ScreenMembersOnMfaCountryCodes;
  /**
   * Sélectionne un code de pays parmi les options disponibles
   * @param payload Les options contenant l'action de sélection du code de pays
   */
  selectCountryCode(payload: SelectCountryCodeOptions): Promise<void>;
  /**
   * Navigates back to the previous screen
   * @param payload Optional custom options to include with the request
   */
  goBack(payload?: CustomOptions): Promise<void>;
}
```
