---
title: "UsernameValidationError"
---

Représente une erreur unique de validation de nom d&#39;utilisateur renvoyée lors de la validation.

Chaque erreur comporte un `code` lisible par une machine et un `message` lisible par un humain.

```ts Example
export interface UsernameValidationError {
  /**
   * Identifiant unique de la règle de validation qui a échoué.
   * 
   * Exemple : `"username-too-short"`, `"username-invalid-characters"`
   */
  code: string;

  /**
   * A human-readable description of the error.
   * 
   * Example: `"Username must be at least 3 characters long."`
   */
  message: string;

  /**
   * The field associated with the validation error, typically "username", "email", or "phone".
   */
  field: string;
}
```


<div id="properties">
  ## Propriétés
</div>

<ParamField body='code' type='string'>

Un identifiant unique pour la règle de validation qui n’a pas été respectée.

Exemple : `"username-too-short"`, `"username-invalid-characters"`

</ParamField>

<ParamField body='field' type='string'>

Le champ associé à l’erreur de validation, généralement "username", "email" ou "phone".

</ParamField>

<ParamField body='message' type='string'>

Une description de l’erreur en langage clair.

Exemple : `"Username must be at least 3 characters long."`
</ParamField>
