PickPhoneOptions
Définit les options pour l’Action « pick-phone ».
Même si l’interface n’affiche qu’un seul numéro de téléphone, cette Action peut être utilisée
dans des scénarios où le serveur doit rediriger vers un écran de sélection du numéro de téléphone.

```ts Example
export interface PickPhoneOptions extends CustomOptions {}
```
