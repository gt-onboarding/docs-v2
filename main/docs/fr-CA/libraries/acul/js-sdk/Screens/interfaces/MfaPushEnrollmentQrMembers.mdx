    <ParamField body="options" type={<span><a href="/docs/libraries/acul/js-sdk/Screens/type-aliases/MfaPollingOptions">MfaPollingOptions</a></span>}>
      <ParamField body="MfaPollingOptions" type="object">
        Options de configuration pour gérer un processus de sondage par notification push d’AMF (MFA).

        Fournissez ces options au démarrage d’une session de sondage pour contrôler la fréquence
        des requêtes vers le point de terminaison du défi et la façon de gérer les réussites ou les échecs.

        <Expandable title="Propriétés">
          <ParamField body="intervalMs?" type="number">
            Intervalle facultatif, en **millisecondes**, entre deux requêtes de sondage consécutives.

            * S’il est omis, l’intervalle interne par défaut de la trousse de développement logiciel (SDK) est utilisé (généralement 5000 ms).
            * Choisissez une valeur qui équilibre la réactivité et les considérations liées aux limites de débit.
          </ParamField>

          <ParamField body="onCompleted()?" type="() => void">
            Rappel facultatif exécuté lorsque le défi push d’AMF (MFA) est approuvé
            avec succès et que le sondage est terminé.

            * Appelé exactement **une seule fois**, après quoi le sondage s’arrête automatiquement.
            * Utile pour faire progresser le flux de connexion (par exemple, en appelant `continue()`).
          </ParamField>

          <ParamField body="onError()?" type="(error) => void">
            Rappel facultatif invoqué si une erreur survient pendant le sondage.

            Reçoit un objet [ULError](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/Error) contenant :

            * `status` — le code d’état HTTP de la requête ayant échoué

            * `responseText` — le corps de la réponse brut, s’il est disponible

            * Appelé une fois par événement d’erreur ; le sondage peut être relancé selon le type d’erreur.

            * À utiliser pour afficher les messages d’erreur ou déclencher une logique personnalisée de nouvelle tentative ou d’abandon.
