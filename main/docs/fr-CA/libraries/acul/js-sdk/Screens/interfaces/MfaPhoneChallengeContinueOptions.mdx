ContinueOptions
Définit les options requises pour poursuivre le processus de défi téléphonique d’AMF (MFA).
Cela consiste généralement à sélectionner la méthode (SMS ou voix) pour recevoir le code de vérification.

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * Le type de défi à effectuer.
   * 'sms' : Envoie le code par message texte.
   * 'voice' : Envoie le code par appel vocal.
   * @default 'sms'
   */
  type: 'sms' | 'voice';
}
```
