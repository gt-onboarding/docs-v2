ScreenMembersOnCustomizedConsent

```ts Example
export interface ScreenMembersOnCustomizedConsent extends ScreenMembers {
  /**
   * An array of `Scope` objects detailing each permission (scope) being requested.
   * These are typically high-level permissions like 'read:profile' or 'openid'.
   * This list should be displayed to the user for their review.
   */
  scopes: Scope[];

  /**
   * Un tableau d'objets `AuthorizationDetail`.
   * Chaque objet fournit des détails granulaires sur les données spécifiques ou
   * les actions que l'application souhaite effectuer, potentiellement liées aux
   * demandes d'autorisation enrichies (RAR) ou à d'autres modèles de permissions granulaires.
   * Cette liste doit être affichée à l'utilisateur pour révision.
   */
  authorizationDetails: AuthorizationDetail[];
}
```
