---
title: "LoginEnrollOptions"
---

Options permettant de poursuivre avec le facteur AMF (MFA) sélectionné

```ts Example
export interface LoginEnrollOptions {
  /** L'action indiquant le facteur à utiliser pour la connexion */
  action: MfaLoginFactorType;
  /** Toutes options personnalisées supplémentaires */
  [key: string]: string | number | boolean | undefined;
}
```

</div>
