---
title: "MfaEmailListMembers"
---

Interface qui définit les méthodes et les propriétés disponibles pour l&#39;écran mfa-email-list.

```ts Example
export interface MfaEmailListMembers extends BaseMembers {
  screen: ScreenMembersOnMfaEmailList;
  /**
   * Sélectionne une adresse courriel inscrite de la liste
   * @param payload Les options contenant l'action de sélection
   */
  selectMfaEmail(payload: SelectMfaEmailOptions): Promise<void>;
  /**
   * Revient à l'écran précédent
   * @param payload Options personnalisées facultatives à inclure avec la requête
   */
  goBack(payload?: CustomOptions): Promise<void>;
}
```
