ContinueOptions

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * Le nouveau surnom que l'utilisateur souhaite attribuer à sa clé de sécurité WebAuthn.
   * Cette valeur sera envoyée au serveur pour mise à jour.
   * Le serveur validera ce surnom (p. ex., longueur, caractères autorisés).
   * @type {string}
   * @example "Primary YubiKey"
   * @example "Home Office Key"
   */
  nickname: string;
}
```

<ParamField body='nickname' type='string'>
Le nouvel alias pour la clé de sécurité WebAuthn.
Il s’agit de la valeur saisie par l’utilisateur dans le champ de saisie.
Elle doit respecter les contraintes de longueur et de caractères définies par le serveur.
</ParamField>
