---
title: "MfaRecoveryCodeEnrollmentMembers"
---

Interface décrivant les membres de l&#39;écran d&#39;inscription au code de récupération pour l&#39;AMF (MFA).

```ts Example
export interface MfaRecoveryCodeEnrollmentMembers extends BaseMembers {
  screen: ScreenMembersOnMfaRecoveryCodeEnrollment;

  /**
   * Déclare que l'utilisateur a enregistré le code de récupération.
   * Cette action est déclenchée lorsque l'utilisateur déclare avoir enregistré le code de récupération.
   * Elle redirige vers l'écran suivant du flux d'authentification.
   *
   * @param {object} payload - Un objet contenant des options personnalisées.
   *
   * @returns {Promise<void>} Une promesse qui se résout lorsque l'action est soumise avec succès.
   * Rejette avec une erreur si la soumission échoue.
   */
  continue(payload: MfaRecoveryCodeEnrollmentContinueOptions): Promise<void>;
}
```
