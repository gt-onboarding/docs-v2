Options pour effectuer des opérations de connexion via les réseaux sociaux

```ts Example
export interface FederatedLoginOptions {
  /** Le nom de la connexion sociale à utiliser */
  connection: string;
  /** Toutes options personnalisées supplémentaires */
  [key: string]: string | number | boolean;
}
```
