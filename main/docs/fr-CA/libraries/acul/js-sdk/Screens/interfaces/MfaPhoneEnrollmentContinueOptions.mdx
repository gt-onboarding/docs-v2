---
title: "MfaPhoneEnrollmentContinueOptions"
---

Options permettant de poursuivre l’inscription à l’AMF (MFA) par téléphone.

Étend `CustomOptions` pour permettre de transmettre des options personnalisées en plus de téléphone et type.

```ts Example
export interface ContinueOptions extends CustomOptions {
  phone: string;
  type: 'sms' | 'voice';
}
```
