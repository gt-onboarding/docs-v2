Options permettant d’effectuer des opérations de connexion

```ts Example
export interface LoginOptions {
  /** Le nom d'utilisateur/courriel avec lequel se connecter */
  username: string;
  /** Le mot de passe pour l'authentification */
  password: string;
  /** Valeur captcha facultative si requise */
  captcha?: string;
  /** Toutes options personnalisées supplémentaires */
  [key: string]: string | number | boolean | undefined;
}
```
