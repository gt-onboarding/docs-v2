UseSecurityKeyOptions

```ts Example
export interface UseSecurityKeyOptions extends CustomOptions {
  /**
   * Facultatif. Si `true`, et si le contexte de l'écran (`screen.showRememberDevice`)
   * indique que la mémorisation de l'appareil est une option, cela signale l'intention
   * de l'utilisateur de mémoriser ce navigateur/appareil pour les authentifications futures.
   * Le SDK soumettra `rememberBrowser=true` dans ce cas.
   * @type {boolean | undefined}
   */
  rememberDevice?: boolean;
}
```

<ParamField body="rememberDevice?" type='boolean'>
Facultatif. Si `true` et que `screen.showRememberDevice` est également `true`,
la Trousse de développement logiciel (SDK) inclura `rememberBrowser=true` dans la soumission du formulaire, indiquant que l'utilisateur a choisi de mémoriser cet appareil.
