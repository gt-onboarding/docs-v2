PickAuthenticatorOptions
Définit les options de l’action &#39;pick-authenticator&#39;.
Cette action permet à l’utilisateur de choisir une autre méthode d’AMF (MFA) si elle est disponible.

```ts Example
export interface PickAuthenticatorOptions extends CustomOptions {}
```
