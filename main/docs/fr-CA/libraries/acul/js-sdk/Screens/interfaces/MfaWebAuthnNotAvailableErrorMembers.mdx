<ParamField body='tryAnotherMethod' type='Promise<void>'>

Permet à l’utilisateur de sélectionner une autre méthode d’AMF (MFA) parce que WebAuthn n’est pas disponible.
Cette action dirige l’utilisateur vers un écran de sélection d’authentificateur où il peut choisir
parmi d’autres facteurs d’AMF disponibles ou déjà inscrits.

Une promesse résolue après l’envoi réussi de l’action « pick-authenticator ».

<div id="throws">
  #### Lève
</div>

Lève une erreur si l’envoi du formulaire échoue (p. ex., problèmes de réseau, erreurs de validation côté serveur).

```typescript Example
// En supposant que 'sdk' soit une instance de MfaWebAuthnNotAvailableError
try {
  await sdk.tryAnotherMethod();
  // En cas de réussite, Auth0 gère la redirection vers l’écran de sélection d’authentificateur.
} catch (error) {
  console.error('Failed to switch to another MFA method:', error);
  // Mettre éventuellement à jour l’interface utilisateur pour informer l’utilisateur de l’échec.
}
```
