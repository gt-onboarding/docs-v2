> **getCurrentThemeOptions**(): [`FlattenedTheme`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

**`Fonctionnalité commune`**

Obtient les options de thème actuelles avec la configuration aplatie provenant du contexte d’image de marque

<div id="returns">
  ## Valeur de retour
</div>

[`FlattenedTheme`](/docs/fr-CA/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

Objet FlattenedTheme qui contient les couleurs, les polices, les bordures, la propriété pageBackground et les configurations de widgets, ou null si aucune image de marque n’est définie
