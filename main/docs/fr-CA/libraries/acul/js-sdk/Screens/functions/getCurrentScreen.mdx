> **getCurrentScreen**(): `string` \| `null`

**`Fonctionnalité commune`**

Récupère le nom de l’écran en cours à partir du contexte d’authentification

<div id="returns">
  ## Valeur de retour
</div>

`string` \| `null`

Le nom de l’écran actuel ou `null` si aucun écran n’est actif
