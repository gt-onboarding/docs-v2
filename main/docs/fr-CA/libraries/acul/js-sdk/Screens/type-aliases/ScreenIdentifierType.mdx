<ParamField body="ScreenIdentifierType" type="email | phone | username">
  Représente les types d&#39;identifiants pouvant être utilisés lors de l&#39;inscription.

  * `'email'`: L&#39;utilisateur fournira une adresse courriel.
  * `'phone'`: L&#39;utilisateur fournira un numéro de téléphone.
  * `'username'`: L&#39;utilisateur choisira un nom d&#39;utilisateur.

  Ce type est généralement utilisé dans les flux de sélection d&#39;identité ou d&#39;inscription
  pour définir quels identifiants sont autorisés et comment ils se comportent.

  ```ts Exemple
  const type: IdentifierType = 'email';
  ```
