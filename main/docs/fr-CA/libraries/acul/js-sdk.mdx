La Trousse de développement logiciel (SDK) JS ACUL d&#39;Auth0 vous permet de travailler avec [Advanced Customization for Universal Login](/docs/fr-CA/customize/login-pages/advanced-customizations).

Elle simplifie l&#39;intégration d&#39;écrans d&#39;authentification (connexion, inscription, authentification sans mot de passe, inscription d&#39;une passkey, etc.) dans vos applications Web, en fournissant les outils nécessaires pour une mise en œuvre fluide.

Pour commencer, installez la Trousse de développement logiciel (SDK) JS ACUL avec npm :

```bash
npm install @auth0/auth0-acul-js
```

Après avoir installé la trousse de développement logiciel (SDK), vous pouvez importer les classes d’écran pour configurer

```bash
// Importation par défaut d'un écran particulier, par ex. : écran login-id
import  LoginId  from '@auth0/auth0-acul-js/login-id'; 

// Named import of any screen
import  { LoginId }  from '@auth0/auth0-acul-js'; 

// Default import of all screens
import  * as Screens  from '@auth0/auth0-acul-js';
```

Explorez les [classes d’écrans et leurs méthodes du SDK JS ACUL](/docs/fr-CA/libraries/acul/js-sdk/Screens/classes/login).

Vous pouvez aussi commencer avec le [Démarrage rapide d’ACUL](/docs/fr-CA/customize/login-pages/advanced-customizations/quickstart).
