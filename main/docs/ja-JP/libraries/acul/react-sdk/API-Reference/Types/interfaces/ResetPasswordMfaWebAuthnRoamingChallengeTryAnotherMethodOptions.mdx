TryAnotherMethodOptions

```ts Example
export interface TryAnotherMethodOptions extends CustomOptions {
  /**
   * オプション。`true`の場合、かつ画面コンテキスト(`screen.showRememberDevice`)が
   * デバイスの記憶がオプションとして利用可能であることを示している場合、
   * ユーザーがこのブラウザ/デバイスを記憶することを意図していることを示します。
   * @type {boolean | undefined}
   */
  rememberDevice?: boolean;
}
```

<ParamField body='rememberDevice?' type='boolean'>
任意。`true` であり、`screen.showRememberDevice` も `true` の場合、
ソフトウェア開発キット (SDK) はフォーム送信時に送信データへ `rememberBrowser=true` を含めます。
</ParamField>
