ResendCodeOptions
説明 ログインのメール確認画面における `resendCode` メソッドのオプション。
この操作は、ユーザーが新しい確認コードの送信をメールで要求したときに使用されます。
たとえば、最初のコードを受信していない場合や、有効期限が切れてしまった場合などです。

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```
