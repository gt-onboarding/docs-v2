別の MFA（多要素認証）方法を試すためのオプションです。

```ts Example
export interface TryAnotherMethodOptions {
  /**
   * 任意の追加カスタムオプション。
   */
  [key: string]: string | number | boolean | undefined;
}
```

  ## インデックス可能
