---
title: "MfaRecoveryCodeEnrollmentMembers"
---

MFA（多要素認証）のリカバリーコード登録画面のメンバーを表すインターフェースです。

```ts Example
export interface MfaRecoveryCodeEnrollmentMembers extends BaseMembers {
  screen: ScreenMembersOnMfaRecoveryCodeEnrollment;

  /**
   * ユーザーがリカバリーコードを保存したことを宣言します。
   * このアクションは、ユーザーがリカバリーコードを保存したことを宣言したときにトリガーされます。
   * 認証フローの次の画面にリダイレクトします。
   *
   * @param {object} payload - カスタムオプションを含むオブジェクト。
   *
   * @returns {Promise<void>} アクションが正常に送信されると解決されるプロミス。
   * 送信が失敗した場合はエラーで拒否されます。
   */
  continue(payload: MfaRecoveryCodeEnrollmentContinueOptions): Promise<void>;
}
```

<ParamField body='continue' type='Promise<void>'>

ユーザーがリカバリーコードを保存したことを示します。
このアクションは、ユーザーがリカバリーコードを保存済みであることを示したときに実行されます。
認証フローの次の画面にリダイレクトします。

アクションの送信が正常に完了すると解決される Promise です。
送信が失敗した場合は Error で拒否されます。
