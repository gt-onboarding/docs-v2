---
title: "MfaPollingResult"
---

[useMfaPolling](/docs/ja-JP/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling) によって返される結果オブジェクトです。

<div id="properties">
  ## プロパティ
</div>

<ParamField body="isRunning" type="boolean">
  MFA（多要素認証）プッシュのポーリング処理が現在実行中かどうかを示します。

  * `true` — ポーリングが実行中で、完了を待機しています。
  * `false` — 完了、手動でのキャンセル、またはコンポーネントのアンマウントにより、
    ポーリングが停止しています。
</ParamField>

<ParamField body="startPolling()" type="void">
  ポーリング処理を開始または再開します。

  * すでにポーリングが実行中の場合、この呼び出しは何も行いません。
  * 過去に停止していた場合、この関数を呼び出すとポーリングループを再開します。
</ParamField>

<ParamField body="stopPolling()" type="void">
  ポーリング処理を即座に停止します。

  * スケジュールされているタイマーや実行中のリクエストをキャンセルします。
  * 複数回呼び出しても安全で、2回目以降の呼び出しは何も行いません。
</ParamField>
