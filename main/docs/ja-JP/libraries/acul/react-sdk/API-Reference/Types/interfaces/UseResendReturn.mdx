---
title: "UseResendReturn"
---

[useResend](/docs/ja-JP/libraries/acul/react-sdk/API-Reference/Hooks/useResend) の戻り値型です。

<div id="properties">
  ## プロパティ
</div>

<ParamField body="remaining" type="number">
  次回の再送信が許可されるまでの残り時間（秒）。
</ParamField>

<ParamField body="disabled" type="boolean">
  再送信アクションが現在無効かどうかを示します。
</ParamField>

<ParamField body="startResend()" type="void">
  許可されている場合は、即座に再送信を開始します。
</ParamField>
