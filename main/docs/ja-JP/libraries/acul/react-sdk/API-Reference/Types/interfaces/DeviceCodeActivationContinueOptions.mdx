---
title: "DeviceCodeActivationContinueOptions"
---

デバイスコード有効化を続行するためのペイロード。

このインターフェースは `CustomOptions` を拡張しており、すべてのカスタムオプションプロパティは
オブジェクトのトップレベルに統合されます。これによりネストを回避でき、
このインターフェースの利用者にとって、より単純な構造になります。

```ts Example
export interface ContinueOptions extends CustomOptions {
  code: string;
}
```

  ## インデックス指定可能
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`
