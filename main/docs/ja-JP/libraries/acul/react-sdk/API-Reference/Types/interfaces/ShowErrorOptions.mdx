---
title: "ShowErrorOptions"
---

ブラウザーの WebAuthn エラーを表示するためのオプション。

```ts Example
export interface ShowErrorOptions {
  /**
   * WebAuthn エラーの詳細情報。
   */
  error: WebAuthnErrorDetails;
  /**
   * Any additional custom options.
   */
  [key: string]: string | number | boolean | undefined | WebAuthnErrorDetails;
}
```

<div id="indexable">
  ## インデックス可能
</div>

[`key`: `string`]: `string` | `number` | `boolean` | [`WebAuthnErrorDetails`](/docs/ja-JP/libraries/acul/react-sdk/API-Reference/Types/interfaces/WebAuthnErrorDetails) | `undefined`

その他の任意のカスタムオプション。

  ## プロパティ
</div>
