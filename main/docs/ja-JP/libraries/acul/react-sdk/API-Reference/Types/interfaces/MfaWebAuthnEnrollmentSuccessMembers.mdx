<ParamField body='continue' type='Promise<void>'>

ユーザーが WebAuthn 登録に成功した後に、認証フローを続行できるようにします。
このメソッドは、`action: "default"` を指定して `/u/mfa-webauthn-enrollment-success` エンドポイントに POST リクエストを送信します。

continue アクションの送信が正常に完了したときに解決される Promise です。
成功すると、ブラウザーは通常、フローの次のステップにリダイレクトされます。

<div id="throws">
  #### 送出される例外
</div>

フォーム送信が失敗した場合（例: ネットワークの問題、無効な state）にはエラーを送出します。

```typescript Example
// 'sdk' が MfaWebAuthnEnrollmentSuccess のインスタンスであると仮定します
try {
  await sdk.continue();
  // ユーザーは次のステップにリダイレクトされます。
} catch (error) {
  console.error("Failed to continue after WebAuthn enrollment:", error);
  // ページがエラー付きで再レンダリングされる場合は sdk.transaction.errors を確認してエラーを処理します。
}
```
