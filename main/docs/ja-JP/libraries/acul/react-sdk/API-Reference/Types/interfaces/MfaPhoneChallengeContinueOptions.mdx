ContinueOptions
電話による MFA（多要素認証）チャレンジ処理を継続するために必要なオプションを定義します。
通常は、認証コードを受信する方法（SMS または音声通話）を選択するために使用されます。

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * 実行するチャレンジのタイプ。
   * 'sms': テキストメッセージ経由でコードを送信します。
   * 'voice': 音声通話経由でコードを送信します。
   * @default 'sms'
   */
  type: 'sms' | 'voice';
}
```

<div id="properties">
  ## プロパティ

<ParamField body='type' type='"sms" | "voice"'>

実施するチャレンジの種別です。
- `sms`: コードをSMSで送信します。
- `voice`: コードを音声通話で送信します。

<div id="default">
  #### 既定値
</div>
