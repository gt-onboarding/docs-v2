---
title: "識別子"
---

サインアップフォームや認証フローで使用する、メール、電話番号、ユーザー名などの識別子フィールドの設定内容を説明します。

```ts Example
export interface Identifier {
  /**
   * 識別子のタイプ（`'email'`、`'phone'`、`'username'` など）。
   *
   * @see IdentifierType
   */
  type: IdentifierType;

  /**
   * Whether this identifier is required for signup.
   */
  required: boolean;
}
```


<div id="example">
  ## 例
</div>

```ts
const emailIdentifier: Identifier = {
  type: 'email',
  required: true
};
```
