ContinueWithCodeOptions
description ログインメール検証画面で使用する `continueWithCode` メソッドのオプション。
この操作は、ユーザーがメールで受信した検証コードを入力して、
認証フローを続行するときに使用されます。

```ts Example
export interface ContinueWithCodeOptions extends CustomOptions {
  /**
   * ユーザーのメールに送信された検証コード。通常、ユーザーがフォームに入力する必要がある短い数値または英数字の文字列です。
   * @type {string}
   * @example "123456"
   * @example "ABCXYZ"
   */
  code: string;
}
```
