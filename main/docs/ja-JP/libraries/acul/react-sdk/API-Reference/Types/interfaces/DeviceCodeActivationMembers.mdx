---
title: "DeviceCodeActivationMembers"
---

デバイスコード有効化画面のメンバーを表すインターフェイスです。

```ts Example
export interface DeviceCodeActivationMembers extends BaseMembers {
  /**
   * ユーザーが入力したデバイスコードを送信します。
   * このアクションは、ユーザーがデバイスに表示されたコードを入力し、フォームを送信したときにトリガーされます。
   *
   * @param {object} payload - ユーザーが入力したコードとカスタムペイロードを含むオブジェクト。
   * @param {string} payload.code - ユーザーが入力したデバイスコード。
   *
   * @returns {Promise<void>} コードが正常に送信されたときに解決される Promise。
   * 送信が失敗した場合はエラーで拒否されます。
   */
  continue(payload: ContinueOptions): Promise<void>;
}
```

<ParamField body='continue' type='Promise<void>'>

ユーザーが入力したデバイスコードを送信します。
このアクションは、ユーザーがデバイスに表示されたコードを入力してフォームを送信したときにトリガーされます。

コードが正常に送信されると解決される Promise を返します。
送信が失敗した場合は、エラーとともに拒否されます。
