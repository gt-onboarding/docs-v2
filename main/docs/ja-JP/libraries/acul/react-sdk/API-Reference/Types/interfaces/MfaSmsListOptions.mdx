MfaSmsListOptions
mfa-sms-list 画面での操作用オプション。

```ts Example
export interface MfaSmsListOptions {
  /**
   * @param {number} index - 選択する電話番号のインデックス。
   */
  index: number;
  /** Any additional custom options */
  [key: string]: string | number | boolean | undefined;
}
```
