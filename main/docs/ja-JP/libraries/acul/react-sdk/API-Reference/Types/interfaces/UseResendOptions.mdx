---
title: "UseResendOptions"
---

[useResend](/docs/ja-JP/libraries/acul/react-sdk/API-Reference/Hooks/useResend) 用のオプション設定です。

<div id="properties">
  ## プロパティ
</div>

<ParamField body="timeoutSeconds?" type="number" default="10">
  再送信が許可されるまでのカウントダウン時間（秒）です。
  デフォルトは `10` です。
</ParamField>

<ParamField body="onTimeout()?" type="void">
  カウントダウンが終了し、再送信アクションが再び利用可能になった時点で呼び出されるコールバックです。
</ParamField>
