---
title: "UsernameValidationResult"
---

ユーザー名を 1 つ以上のルールで検証した結果。

```ts Example
export interface UsernameValidationResult {
  /**
   * Indicates whether the username passed all validation rules.
   */
  isValid: boolean;

  /**
   * 失敗した個々のルールを表す {@link UsernameValidationError} オブジェクトの配列（存在する場合）。
   *
   * {@link UsernameValidationResult.isValid} が `true` の場合、この配列は空です。
   */
  errors: UsernameValidationError[];
}
```
