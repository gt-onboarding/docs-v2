PickPhoneOptions
&#39;pick-phone&#39; Action のオプションを定義します。
インターフェース上には電話番号は 1 つしか表示されませんが、この Action は、
サーバーが電話番号の選択画面へリダイレクトする必要があるシナリオで使用される場合があります。

```ts Example
export interface PickPhoneOptions extends CustomOptions {}
```


<div id="indexable">
  ## インデックス可能型
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`
