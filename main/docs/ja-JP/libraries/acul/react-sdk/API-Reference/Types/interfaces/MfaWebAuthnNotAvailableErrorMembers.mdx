<div id="methods">
  ## メソッド
</div>

<ParamField body='tryAnotherMethod' type='Promise<void>'>

WebAuthn が利用できない場合に、ユーザーが別の MFA（多要素認証）手段を選択できるようにします。
このアクションは、ユーザーを認証器選択画面に遷移させ、利用可能または登録済みの他の MFA 要素から選択できるようにします。

`'pick-authenticator'` アクションが正常に送信されると解決される Promise です。

<div id="throws">
  #### スロー
</div>

フォーム送信が失敗した場合（例: ネットワークの問題、サーバー側のバリデーションエラー）にエラーをスローします。

```typescript Example
// 'sdk' は MfaWebAuthnNotAvailableError のインスタンスであると仮定します
try {
  await sdk.tryAnotherMethod();
  // 成功時には、Auth0 が認証器選択画面へのリダイレクトを処理します。
} catch (error) {
  console.error('Failed to switch to another MFA method:', error);
  // 必要に応じて UI を更新し、失敗したことをユーザーに通知します。
}
```
