---
title: "ContinueWithOrganizationNameOptions"
---

選択した組織名を使用して続行するためのオプション。

```ts Example
export interface ContinueWithOrganizationNameOptions {
  /**
   * 組織の名前。
   */
  organizationName: string;
  /**
   * その他の追加カスタムオプション。
   */
  [key: string]: string | number | boolean | undefined;
}
```

  ## インデックス可能
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`

追加のカスタムオプションです。

  ## プロパティ
</div>
