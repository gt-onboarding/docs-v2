---
title: "型エイリアス: ScreenIdentifierType"
---

<ParamField body="ScreenIdentifierType" type='"email" | "phone" | "username"'>
  

サインアップ時に使用できるユーザー識別子の種別を表します。

- `'email'`: ユーザーがメールアドレスを入力します。
- `'phone'`: ユーザーが電話番号を入力します。
- `'username'`: ユーザーがユーザー名を選択します。

この型は通常、アイデンティティの選択やサインアップフローで使用され、どの識別子を有効にし、どのように動作させるかを定義します。

```ts Example
const type: IdentifierType = 'email';
```

</ParamField>
