MfaEnrollResult

```typescript Example
import MfaEnrollResult from '@auth0/auth0-acul-js/mfa-enroll-result';

const mfaEnrollResultScreen = new MfaEnrollResult();

// 画面データにアクセスする
const enrollmentStatus = mfaEnrollResultScreen.screen.data?.status;
const pageTitle = mfaEnrollResultScreen.screen.texts?.title;

console.log(`MFA登録ステータス: ${enrollmentStatus}`);
console.log(`ページタイトル: ${pageTitle}`);
```


<div id="constructors">
  ## コンストラクター
</div>

<ParamField body="MfaEnrollResult" type="Constructor">

MfaEnrollResult 画面マネージャーのインスタンスを作成します。
`BaseContext` を初期化し、この画面用に調整された `ScreenOverride` のインスタンスを
`screen` プロパティに設定します。

<div id="throws">
  #### スロー
</div>

Universal Login コンテキストが利用できない場合、または
コンテキスト内の現在の画面名が `MfaEnrollResult.screenIdentifier` と一致しない場合にスローされます。

</ParamField>
