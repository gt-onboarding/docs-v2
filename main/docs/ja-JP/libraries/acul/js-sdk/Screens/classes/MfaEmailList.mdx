---
title: "MfaEmailList"
---

mfa-email-list 画面の機能を実装するクラスです
この画面では、ユーザーが MFA（多要素認証）に使用する登録済みのメールアドレスを選択できます

```ts Example
import MfaEmailList from '@auth0/auth0-acul-js/mfa-email-list';

const mfaEmailList = new MfaEmailList();
await mfaEmailList.selectMfaEmail({
  index: 0 // デモのため、最初のインデックスを選択
});
```
