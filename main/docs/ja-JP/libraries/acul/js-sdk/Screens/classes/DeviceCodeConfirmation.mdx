デバイスコード確認画面の機能を実装するクラスです。
この画面は、ユーザーがデバイスコードを確認する必要がある際に表示されます。

```ts Example
import DeviceCodeConfirmation from '@auth0/auth0-acul-js/device-code-confirmation';
const deviceCodeConfirmation = new DeviceCodeConfirmation();
```
