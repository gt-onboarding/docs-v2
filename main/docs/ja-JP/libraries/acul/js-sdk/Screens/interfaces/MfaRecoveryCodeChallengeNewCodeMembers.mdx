MfaRecoveryCodeChallengeNewCodeMembers

````ts Example
export interface MfaRecoveryCodeChallengeNewCodeMembers extends BaseMembers {
  /**
   * Access to the specific properties and data of the 'mfa-recovery-code-challenge-new-code' screen.
   * Includes the `textCode` which is the recovery code to be displayed.
   * @type {ScreenMembersOnMfaRecoveryCodeChallengeNewCode}
   */
  screen: ScreenMembersOnMfaRecoveryCodeChallengeNewCode;

  /**
   * ユーザーが新しいリカバリーコードを保存したことを確認し、認証フローを続行します。
   * このメソッドは、ユーザーが表示されたリカバリーコードを安全に保存したことを示した後に呼び出す必要があります。
   * 次のステップに進むために Auth0 サーバーに確認を送信します。
   *
   * @param {ContinueOptions} [payload] - リクエストに含めるオプションのカスタムデータ。
   * @returns {Promise<void>} 確認が正常に送信されたときに解決される Promise。
   * @throws {Error} 送信が失敗した場合（例: ネットワークの問題、無効な状態）にエラーをスローします。
   *
   * @example
   * ```typescript
   * import MfaRecoveryCodeChallengeNewCode from '@auth0/auth0-acul-js/mfa-recovery-code-challenge-new-code';
   *
   * const screenManager = new MfaRecoveryCodeChallengeNewCode();
   *
   * // ユーザーがコードを保存したことを確認するチェックボックスをオンにしたと仮定
   * try {
   *   await screenManager.continue();
   *   // 成功時に次の画面にリダイレクト
   * } catch (error) {
   *   console.error("Failed to confirm recovery code saved:", error);
   *   // エラーを処理、例: screenManager.transaction.errors からメッセージを表示
   * }
   * ```
   */
  continue(payload?: ContinueOptions): Promise<void>;
}
````

<ParamField body='continue' type='Promise<void>'>

ユーザーが新しいリカバリーコードを保存したことを確認し、認証フローを継続します。
このメソッドは、ユーザーが表示されたリカバリーコードを安全に保管したことを示した後に呼び出す必要があります。
次のステップに進むための確認を Auth0 サーバーに送信します。

確認が正常に送信されると解決される Promise です。

<div id="throws">
  #### スロー
</div>

送信に失敗した場合（例: ネットワークの問題、state が不正な場合）にエラーをスローします。

```typescript Example
import MfaRecoveryCodeChallengeNewCode from '@auth0/auth0-acul-js/mfa-recovery-code-challenge-new-code';

const screenManager = new MfaRecoveryCodeChallengeNewCode();

// ユーザーがコードを保存したことを確認するチェックボックスをオンにしたと仮定します
try {
  await screenManager.continue();
  // 成功時に次の画面へリダイレクト
} catch (error) {
  console.error("Failed to confirm recovery code saved:", error);
  // エラーを処理（例: screenManager.transaction.errors からのメッセージを表示）
}
```
