PickAuthenticatorOptions
&#39;pick-authenticator&#39; Action のオプションを定義します。
これにより、利用可能な場合には、ユーザーが別の MFA（多要素認証）手段を選択できるようになります。

```ts Example
export interface PickAuthenticatorOptions extends CustomOptions {}
```
