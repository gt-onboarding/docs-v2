---
title: "RedeemTicketMembers"
---

Redeem Ticket 画面で使用できるデータを表すインターフェースです。

````ts Example
export interface RedeemTicketMembers extends BaseMembers {
  client: ClientMembers;
  organization: OrganizationMembers;
  prompt: PromptMembers;
  screen: ScreenMembers;
  transaction: TransactionMembers;

  /**
   * redeem-ticket画面でデフォルトのアクションを実行します。通常、フローの続行を伴います。
   * @param {CustomOptions} [payload] - リクエストに含めるオプションのカスタムオプション。
   * @returns {Promise<void>} アクションが完了したときに解決されるPromise。
   * @throws {Error} 操作が失敗した場合。
   * @example
   * ```typescript
   * import RedeemTicket from '@auth0/auth0-acul-js/redeem-ticket';
   *
   * const redeemTicket = new RedeemTicket();
   * await redeemTicket.continue();
   * ```
   */
  continue(payload?: CustomOptions): Promise<void>;
}
````

  ## メソッド
</div>

<ParamField body='continue' type='Promise<void>'>

`redeem-ticket` 画面でデフォルトのアクションを実行します。通常はフローを継続します。

アクションが完了すると解決される Promise です。

<div id="throws">
  #### スロー
</div>

操作が失敗した場合にスローされます。

```typescript Example
import RedeemTicket from '@auth0/auth0-acul-js/redeem-ticket';

const redeemTicket = new RedeemTicket();
await redeemTicket.continue();
```
