ContinueWithCodeOptions
description ログイン用メール確認コード画面における `continueWithCode` メソッドのオプションを表します。
このオペレーションは、ユーザーがメールで受け取った認証コードを送信して、
認証フローを継続する際に使用されます。

```ts Example
export interface ContinueWithCodeOptions extends CustomOptions {
  /**
   * ユーザーのメールに送信された検証コード。通常、ユーザーがフォームに入力する必要がある短い数値または英数字の文字列です。
   * @type {string}
   * @example "123456"
   * @example "ABCXYZ"
   */
  code: string;
}
```
