PickPhoneOptions
`pick-phone` Action のためのオプションを定義します。
インターフェース上では電話番号は 1 件しか表示されませんが、この Action は、
サーバー側でユーザーを電話番号選択画面にリダイレクトする必要があるシナリオで使用されることがあります。

```ts Example
export interface PickPhoneOptions extends CustomOptions {}
```
