---
title: "識別子"
---

サインアップフォームや認証フローで使用する、メール、電話番号、ユーザー名などの識別子フィールドを記述します。

```ts Example
export interface Identifier {
  /**
   * 識別子のタイプ（`'email'`、`'phone'`、`'username'` など）。
   *
   * @see IdentifierType
   */
  type: IdentifierType;

  /**
   * Whether this identifier is required for signup.
   */
  required: boolean;
}
```


<div id="example">
  ## 例
</div>

```ts
const emailIdentifier: Identifier = {
  type: 'email',
  required: true
};
```


<div id="properties">
  ## プロパティ
</div>

<ParamField body='required' type='boolean'>

この識別子がサインアップに必須かどうかを示します。

</ParamField>

<ParamField body='type' type={<span><a href="/docs/libraries/acul/js-sdk/Screens/type-aliases/ScreenIdentifierType">ScreenIdentifierType</a></span>}>

識別子のタイプです。たとえば、`email`、`phone`、`username` などです。

Type
</ParamField>
