VerifyPlatformAuthenticatorOptions

```ts Example
export interface VerifyPlatformAuthenticatorOptions extends CustomOptions {
  /**
   * オプション。trueかつ`screen.showRememberDevice`がtrueの場合、ユーザーがこのデバイスを記憶することを選択したことを示します。通常30日間、以降のログイン時のMFA（多要素認証）をバイパスします。
   * @type {boolean}
   */
  rememberDevice?: boolean;
}
```


<div id="indexable">
  ## インデックス指定可能
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`

<div id="properties">
  ## プロパティ
</div>

<ParamField body='rememberDevice?' type='boolean'>
省略可。true の場合、今後の MFA（多要素認証）チャレンジの際にブラウザを記憶します。
`rememberBrowser` フォームフィールドに対応します。これは、`screen.showRememberDevice` が true の場合にのみ適用されます。
</ParamField>
