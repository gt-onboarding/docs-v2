---
title: "ResendCodePayloadOptions"
---

メールで送信されるコードを再送信するためのオプション

```ts Example
export interface ResendCodeOptions {
  /** 追加のカスタムオプション */
  [key: string]: string | number | boolean | undefined;
}
```


<div id="indexable">
  ## インデックス可能
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`

任意のカスタムオプションを追加できます
