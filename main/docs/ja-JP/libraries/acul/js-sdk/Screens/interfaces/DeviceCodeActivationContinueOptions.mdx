---
title: "DeviceCodeActivationContinueOptions"
---

デバイスコード有効化処理を続行するためのペイロードです。

このインターフェースは `CustomOptions` を拡張しており、すべてのカスタムオプションプロパティは
オブジェクトのトップレベルにマージされます。これによりネストを避け、このインターフェースの
利用者にとってよりシンプルな構造になります。

```ts Example
export interface ContinueOptions extends CustomOptions {
  code: string;
}
```

  ## インデックス可能
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`
