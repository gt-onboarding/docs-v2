UseSecurityKeyOptions

```ts Example
export interface UseSecurityKeyOptions extends CustomOptions {
  /**
   * オプション。`true` の場合、かつ画面コンテキスト (`screen.showRememberDevice`)
   * でデバイスを記憶するオプションが利用可能であることが示されている場合、
   * 今後の認証のためにこのブラウザ/デバイスを記憶するというユーザーの意図を示します。
   * この場合、SDK は `rememberBrowser=true` を送信します。
   * @type {boolean | undefined}
   */
  rememberDevice?: boolean;
}
```

<ParamField body='rememberDevice?' type='boolean'>
省略可能。`true` で、かつ `screen.showRememberDevice` も `true` の場合、
ソフトウェア開発キット (SDK) はフォーム送信時に `rememberBrowser=true` を含めて送信し、ユーザーがこのデバイスを記憶させることを選択したことを示します。
</ParamField>
