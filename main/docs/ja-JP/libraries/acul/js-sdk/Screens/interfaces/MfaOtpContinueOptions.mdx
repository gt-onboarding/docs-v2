---
title: "MfaOtpContinueOptions"
---

MFA（多要素認証）の OTP 登録コードで続行するためのオプションです。

```ts Example
export interface ContinueOptions {
  /**
   * ユーザーが入力したOTPコード。
   */
  code: string;

  /**
   * その他の追加カスタムオプション。
   */
  [key: string]: string | number | boolean | undefined;
}
```


<div id="indexable">
  ## インデックス指定可能
</div>

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`

任意の追加カスタムオプション。

<div id="properties">
  ## プロパティ
</div>

<ParamField body='code' type='string'>

ユーザーが入力したワンタイムパスワード (OTP) のコード。
</ParamField>
