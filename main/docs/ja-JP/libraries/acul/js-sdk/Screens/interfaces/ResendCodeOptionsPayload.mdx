ResendCodeOptions
description ログイン用メール確認画面の `resendCode` メソッド用オプション。
この操作は、ユーザーが新しい認証コードを自分のメールアドレス宛てに再送信するよう要求したときに使用されます。
たとえば、最初のコードを受け取っていない場合や、有効期限が切れてしまった場合などです。

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```
