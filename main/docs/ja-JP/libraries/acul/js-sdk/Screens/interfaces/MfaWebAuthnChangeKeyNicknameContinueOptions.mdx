ContinueOptions

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * ユーザーがWebAuthnセキュリティキーに割り当てる新しいニックネーム。
   * この値はサーバーに送信されて更新されます。
   * サーバーはこのニックネームを検証します（例：長さ、使用可能な文字）。
   * @type {string}
   * @example "Primary YubiKey"
   * @example "Home Office Key"
   */
  nickname: string;
}
```
