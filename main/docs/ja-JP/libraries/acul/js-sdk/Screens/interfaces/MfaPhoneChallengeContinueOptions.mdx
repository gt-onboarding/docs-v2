ContinueOptions
電話番号を使った MFA（多要素認証）チャレンジ処理を継続するために必要なオプションを定義します。
通常は、認証コードを受け取る手段（SMS または音声通話）の選択が含まれます。

```ts Example
export interface ContinueOptions extends CustomOptions {
  /**
   * 実行するチャレンジのタイプ。
   * 'sms': テキストメッセージ経由でコードを送信します。
   * 'voice': 音声通話経由でコードを送信します。
   * @default 'sms'
   */
  type: 'sms' | 'voice';
}
```
