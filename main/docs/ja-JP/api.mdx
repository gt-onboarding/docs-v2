  ## ページネーション

  Public Cloud テナントの場合、Management API エンドポイントはレスポンスごとに最大 50 件まで結果を返します。1 回のレスポンスで返せる件数を超える結果がある場合は、より多くの結果を取得するために、リクエストに `page` と `per_page` パラメーターを含めてください。

  アプリケーションの更新方法については、[Management API v2 エンドポイントのページネーションされたクエリへの移行](/docs/ja-JP/troubleshoot/product-lifecycle/past-migrations/migrate-to-paginated-queries)を参照してください。
