Auth0 Dashboard を使用すると、テナントのアクセス制御リスト (ACL) ルールの一覧を表示し、各ルールを個別に有効化または無効化できます。

<div id="view-list-of-rules">
  ## ルールの一覧を表示する
</div>

既存のルールの一覧を表示するには、[Dashboard &gt; Security &gt; Access Control](https://manage.auth0.com/#/security/access-control) に移動します。

| 列名 | 説明 | 関連する API レスポンスフィールド |
| --- | --- | --- |
| Priority | ルールが評価される順序を示す番号 | `priority` |
| Name | ルールの目的または機能の説明 | `description` |
| Action | ルールが実行する[アクション](/docs/ja-JP/secure/tenant-access-control-list/reference#actions)。 | `action` |
| Status | ルールが有効か無効かを示すステータス値。 | `active` |
| Last Updated | ルールが最後に更新された日時（タイムスタンプ）。 | `updated_at` |

![Auth0 Dashboard で Tenant ACL ルールの一覧を表示している画面](/docs/images/cdy7uua7fh8z/4093f9a39aba48bdf8d579c134dc4da5/Screenshot_2025-09-03_at_12.33.47.png)

<div id="enable-or-disable-a-rule">
  ## ルールを有効または無効にする
</div>

特定のルールは、その横にあるボタンを切り替えることで有効または無効にできます。

**Status** フィールドには、ルールの状態が `Active` または `Disabled` と表示されます。
